# CR7 Website — Project Workflow & Instructions

> This file governs how all data is collected, stored, and used in the CR7 website project.
> **Read before taking any action.**

---

## ✅ Core Rule

**All research and data goes into a single MD file first.**
No Excel. No random files. Everything organized and planned before it moves to the website.

---

## 📁 File Structure

```
CR7 Website Building Demo (Testing)/
│
├── PROJECT_WORKFLOW.md         ← This file. Rules + instructions.
│
├── CR7_Goals_Data.md           ← All CR7 career goals (971 goals, growing)
│
├── [NEXT] CR7_Stats_Overview.md      ← Career stats, records, awards
├── [NEXT] CR7_Club_History.md        ← Each club: period, trophies, highlights
├── [NEXT] CR7_Personal_Info.md       ← Bio, family, personal facts
├── [NEXT] CR7_Awards.md             ← Ballon d'Or, records, individual awards
├── [NEXT] CR7_Photos_Index.md       ← Index of uploaded photos + sourced photos
│
└── /photos/                         ← All photos used in the website
```

---

## 📋 How We Work — Step by Step

### Step 1 — Research First
- Always search multiple sources before writing anything
- Cross-check facts (at least 2 sources)
- Use: roadto1000goals.com · messivsronaldo.app · Wikipedia · transfermarkt · fbref.com

### Step 2 — Write to MD File
- All data goes into the correct MD file
- Use tables for structured data (goals, stats, awards)
- Add the source at the bottom of every section

### Step 3 — Review
- Check for errors, missing data, wrong facts
- Mark anything uncertain with ⚠️

### Step 4 — Website
- Only after the MD file is complete and reviewed does data move to the website

---

## 📸 Photos

- User-uploaded photos: stored in /photos/ folder
- Internet-sourced photos: must be credited and copyright-safe
- All photos indexed in CR7_Photos_Index.md before use

---

## ✏️ Updating Goals Data

When CR7 scores a new goal:
1. Open `CR7_Goals_Data.md`
2. Add a new row to the **Goals #900–971 table** (increment the number)
3. Update the **Career Overview** total at the top
4. Update the **Goals to 1,000** counter

---

## ❌ What We Don't Do

- Don't create random files without a plan
- Don't put data directly into the website without organizing it in MD first
- Don't add unverified stats without a source
- Don't use Excel (switched to MD for this project)

---

## 🔄 TR: (Try Again) Protocol

If any data seems wrong or doesn't make sense, mark it with **TR:** and we will re-search and correct it before it goes to the website.
